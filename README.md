
# 🎥 VlogMe

> “Capture your moments. Speak your story. Vlog with VlogMe.” 🖤

**VlogMe** is a modern, minimal Android app for personal **vlogging**. Record videos, add a title and mood, and revisit your memories with ease — all in a beautiful black-and-white UI built using **Jetpack Compose** and **CameraX**.

---

## ✨ Features

- 🎬 **Record short videos** with audio using CameraX
- 📝 **Add custom title & mood** after recording
- 📁 **Save videos** to device storage via MediaStore
- 📺 **View vlogs** with built-in video player
- 🧹 **Delete vlogs** anytime from your list
- 🖤 **Dark theme** with clean, minimal design

---

## 🛠️ Tech Stack

- **Jetpack Compose** — UI Toolkit
- **CameraX** — for video recording
- **MediaStore API** — to save videos on device
- **Room DB** — to store vlog metadata
- **MVVM Architecture** — for clean code structure
- **Hilt** — for dependency injection
